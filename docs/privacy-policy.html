プライバシーポリシー

Throttling VPN（dev.kkhouse.throttlingvpn）

1. はじめに

本アプリ「Throttling VPN」（以下「本アプリ」）は、開発者 kk-house（以下「開発者」）が提供するネットワーク速度制限テスト用の VPN アプリです。本プライバシーポリシーは、本アプリにおける情報の取り扱いについて説明します。

2. 収集する情報

本アプリは、ユーザーの個人情報を一切収集・送信しません。

  • アカウント登録は不要です
  • アナリティクスやトラッキングツールは使用していません
  • 広告 SDK は含まれていません
  • 外部サーバーへのデータ送信は行いません

3. VPN サービスについて

本アプリは Android の VpnService API を使用します。VPN 接続はすべてデバイス上でローカルに処理され、ネットワークトラフィックが外部のプロキシやサーバーを経由することはありません。

VPN 機能は、選択したアプリの通信速度を制限（スロットリング）する目的でのみ使用され、通信内容の監視・記録・改変は行いません。

4. データの保存

以下の設定情報がデバイス内にのみ保存されます。

  • スロットリング対象として選択したアプリの一覧
  • 速度設定値

これらのデータはデバイス外に送信されることはなく、アプリのアンインストール時に削除されます。

5. 第三者への提供

本アプリはユーザーデータを収集しないため、第三者への提供・共有・販売は一切ありません。

6. 子供のプライバシー

本アプリは13歳未満の子供から意図的に個人情報を収集することはありません。

7. ポリシーの変更

本プライバシーポリシーは予告なく変更される場合があります。変更があった場合は本ページを更新します。

8. お問い合わせ

本ポリシーに関するお問い合わせは、以下の GitHub リポジトリの Issue よりお願いいたします。

https://github.com/kk-house-777/throttling-vpn-app/issues

Privacy Policy

Throttling VPN (dev.kkhouse.throttlingvpn)

1. Introduction

Throttling VPN ("the App") is a network speed-throttling VPN tool provided by kk-house ("the Developer"). This Privacy Policy describes how information is handled within the App.

2. Information We Collect

The App does not collect or transmit any personal information.

  • No account registration is required
  • No analytics or tracking tools are used
  • No advertising SDKs are included
  • No data is sent to external servers

3. VPN Service

The App uses the Android VpnService API. All VPN connections are processed locally on the device. Network traffic is never routed through any external proxy or server.

The VPN functionality is used solely to throttle the network speed of selected apps. It does not monitor, log, or modify the contents of network traffic.

4. Data Storage

The following settings are stored locally on the device only:

  • List of apps selected for throttling
  • Speed configuration values

This data is never transmitted outside the device and is deleted when the App is uninstalled.

5. Third-Party Sharing

Since the App does not collect user data, there is no sharing, selling, or disclosure of data to third parties.

6. Children's Privacy

The App does not knowingly collect personal information from children under the age of 13.

7. Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be reflected on this page.

8. Contact

For questions about this policy, please open an issue on the GitHub repository:

https://github.com/kk-house-777/throttling-vpn-app/issues
